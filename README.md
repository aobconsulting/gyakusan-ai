# 🧭 逆算AI — ゴール逆算型タスクスケジューラー

> 「何から始めればいいか分からない」を解消する、AIスケジュール生成ツール

[![License: MIT](https://img.shields.io/badge/License-MIT-orange.svg)](https://opensource.org/licenses/MIT)
[![GAS](https://img.shields.io/badge/Google%20Apps%20Script-✓-blue.svg)](https://script.google.com/)
[![Gemini](https://img.shields.io/badge/Gemini%20API-2.5%20flash-brightgreen.svg)](https://aistudio.google.com/)

## ✨ 特徴

- **AIヒアリング** — 選択肢タップ方式で「本当のゴール」を明確化（最大5問・AIが十分な情報が集まったと判断した時点でゴール選択へ）
- **推奨ゴール日表示** — タスク規模に応じた推奨ゴール日を自動計算・提案（強制ではなく参考として）
- **逆算スケジュール生成** — ゴール日から逆算したマイルストーンを自動生成
- **プレビュー＆編集** — 生成されたスケジュールを確認し、日付・時間の変更やタスク削除が可能
- **保存先選択** — スプレッドシート / Googleカレンダー / 両方から選べる
- **スマートリマインダー** — カレンダー保存時、参集系タスクには1週間前アラートを自動設定
- **コーヒーブレイクUI** — APIビジー時にアニメーション付きで自動リトライ（エラーメッセージなし）
- **スケジュール履歴保存** — 生成するたびにシートが残る（上書きなし）
- **組織対応** — 複数人で使うとユーザー名が自動付与
- **具体⇔抽象トレーニング** — 使うだけで思考の解像度が上がる
- **仕事・事業・プライベート全対応** — 上司の指示も起業も結婚式の準備も

## 🚀 セットアップ

### 必要なもの

- Google アカウント（Workspace 推奨）
- Gemini API キー（[Google AI Studio](https://aistudio.google.com/apikey) で無料取得）

### 手順

1. **Gemini API キーを取得**  
   [Google AI Studio](https://aistudio.google.com/apikey) にアクセス → APIキーをコピー

2. **スプレッドシートを新規作成**  
   Google ドライブで「新規」→「Google スプレッドシート」

3. **Apps Script エディタを開く**  
   「拡張機能」→「Apps Script」

4. **コードを貼り付ける**  
   - `コード.gs` の中身を全削除 → `Code.gs` の内容を全てコピペ  
   - 左の「＋」→「HTML」→ ファイル名を `ui` に変更 → `ui.html` の内容を全てコピペ
   - `manual.html` も同じ要領で追加（ファイル名は `manual`）

5. **APIキーをスクリプトプロパティに設定**  
   「プロジェクトの設定（⚙️）」→「スクリプトプロパティ」→「スクリプトプロパティを追加（または編集）」  
   プロパティ名：`GEMINI_API_KEY` / 値：取得したAPIキー

6. **スプレッドシートを再読み込み**（F5）

7. **メニュー「🧭 逆算AI」→「🚀 逆算AIを起動」** 🎉

### 複数人で使う場合（組織利用）

「拡張機能」→「Apps Script」→「デプロイ」→「新しいデプロイ」でウェブアプリとしてデプロイする際、以下の設定が必要です：

| 設定項目 | 値 |
|---|---|
| 次のユーザーとして実行 | **⚠️ ウェブアプリにアクセスしているユーザー** |
| アクセスできるユーザー | 全員（または同じドメイン内） |

> **「自分」のままにすると全員のカレンダー登録が管理者のカレンダーに書き込まれます。**  
> 「アクセスしているユーザー」にすることで、各メンバー自身のGoogleカレンダーに登録されます。  
> 初回アクセス時にGoogle権限許可画面が表示されます（「許可」で以降は非表示）。

> 📖 詳しい手順・図解つきは **manual.html** をご覧ください。
> 開き方：このリポジトリの `manual.html` をクリック → ファイル上部の **「Raw」ボタン** をクリック → ブラウザにHTMLマニュアルが表示されます。
> またはリポジトリをダウンロード/クローンして、`manual.html` をブラウザで直接開いてもOKです。

## ⚡ AIがビジー状態になったときは

ヒアリング中に「AIがコーヒーブレイク中...」というアニメーションが表示されることがあります。  
これは逆算AIのバグや不具合ではなく、**Gemini API（Google提供）側の一時的な混雑**によるものです。

逆算AIは内部でGemini APIの無料枠を使用しています。無料枠には以下の利用制限があります：

| 制限 | 上限 |
|---|---|
| 1分あたりのリクエスト数 | 10回/分 |
| 1日あたりのリクエスト数 | 500回/日 |

短時間に集中してテストや利用を行うと上限に達することがあります。  
ビジー状態になった場合、逆算AIは自動的に待機してリトライします（最大4回）。  
それでも復帰しない場合は、1〜2分待ってから再度お試しください。

> **1日の上限に達した場合は翌日にリセットされます。**

### ビジー状態が頻繁に起きる場合：有料プランへの移行

頻繁にビジー状態になる場合は、Gemini APIの**従量課金制（Pay-as-you-go）**への移行を検討してください。

**移行手順：**  
[Google AI Studio](https://aistudio.google.com/) → 「請求を設定」→ Google Cloudプロジェクトに課金を有効化

**有料プランのメリット：**
- レート制限が大幅に緩和（商用レベルのリクエスト数）
- 入力データがGoogleのモデル学習に使用されない（プライバシー向上）
- 安定した応答速度

**料金の目安（Gemini 2.5 Flash）：**
- 入力：$0.30 / 100万トークン
- 出力：$2.50 / 100万トークン
- 逆算AIの1セッション（ヒアリング〜スケジュール生成）は概ね $0.01〜0.03 程度

> 最新の料金は [Gemini API 料金ページ](https://ai.google.dev/gemini-api/docs/pricing) をご確認ください。
